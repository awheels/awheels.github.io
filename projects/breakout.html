Breakout Video Game

For the Capstone Project for my Masters in Mathematics for Teachers program I needed to develop an enrichment unit that could be used in a high school classroom. I created an activity that walked students through the development of the classic Breakout arcade game. The game was created using Ruby and the Gosu 2D game development library.
