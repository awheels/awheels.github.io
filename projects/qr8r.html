Curator (QR8R) Final Project

I completed this image curating application for my final project while completing the Lighthouse Labs Web Development program. The application allows users to setup private and public galleries that save photographs that you post to instagram based on specific hashtags. The project was completed over the course of one week with three other group members. It uses Ruby and the Ruby on Rails framework.
